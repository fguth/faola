f.aola is a helpful tool for styling buttons in an easy and fast way. You type in a color and we give you the css (no images!) to apply all over your project. Yes, it's that simple.

Type in the base color to be applied on the button style. Only hex values are accepted.

You can choose the roundness of your button from one of the options on the side. We do not allow custom values yet.

Choose one of the options of buttons. Each one have its own hover and active states style.

Done?

All you need to do now is to load the stylesheet on your project and apply the class "faola" wherever you want to see your button styled.
